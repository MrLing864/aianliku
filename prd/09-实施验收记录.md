## 3. 已验证的生产行为

- 首页、案例列表、体检页、后台登录和 robots 均返回正常响应；
- 后台未登录访问被重定向，管理员登录后可进入治理工作区；
- DeepSeek 动态追问使用真实服务返回结构化 AI 结果；
- 完整报告提交前分别取得报告生成同意、隐私处理同意，并记录告知版本与同意时间；营销联系保持独立可选且默认关闭；
- 私密报告响应包含 `private, no-cache, no-store` 和 `X-Robots-Tag: noindex`；
- MongoDB 未配置时，联系、预约和完整报告不会伪造成功，明确返回 `503`；
- 最近一次检查未发现 EdgeOne Pages 运行时错误。

- 合并案例旧 Slug 的永久跳转与 Canonical；
- 案例乐观锁、版本快照、归档页和删除状态；
- 体检持久任务、重试、ROI 版本和级联删除；
- 原始问答查看、案例合并及联系方式导出的二次验证与审计。

## 5. 尚未满足的正式上线门槛

以下项目依赖运营主体、第三方账号或正式数据，当前不得标记为通过：

1. 配置生产 `MONGODB_URI`（MongoDB Atlas 标准连接串），运行 `npm run db:setup` 和 `npm run db:seed`，再执行并发唯一索引、Atlas Search/Vector Search、事务及恢复测试；
2. 导入并人工核验至少 100 条正式案例，覆盖四种结果、三种可信度、多来源和重复样本；
3. 配置手机号必填与格式校验，验证无效号码被拒、重发限流（页面内交付，不再通过邮件）；
4. 配置腾讯云 COS，验证后台授权与五分钟内短期签名快照；
5. 完成生产备份恢复与删除墓碑重放演练；
6. 在中国大陆至少三种网络环境完成性能和可用性实测；
7. 由实际运营主体完成隐私、长期保存、跨境数据流、第三方服务及内容版权的正式合规评审；
8. 配置 `OPS_ALERT_WEBHOOK_URL`，验证关键错误、报告生成失败和删除失败告警的实际送达；数据库连接异常已在后台运行状态页展示，仍需接入外部巡检。
